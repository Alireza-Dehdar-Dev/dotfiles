# Personal Linux Dotfiles

This repository contains my personal Linux dotfiles, designed to create a fast, efficient, and consistent development environment. It includes configurations for various tools and utilities, optimized for a clean workflow across different machines.

## Included Configurations

- **i3**: Custom configuration for a tiling window manager.
- **Neovim**: Setup with custom plugins and settings for development.
- **Zsh**: Enhanced Zsh shell setup with useful aliases and themes.
- **Alacritty**: Terminal configuration for a minimal and responsive experience.
- **Scripts**: Utility scripts for automation and system setup.

## Features

- **Easy Setup**: Fast installation via symlinks, designed to quickly set up your development environment on any new machine.
- **Cross-Machine Compatibility**: Consistent environment setup across multiple devices.
- **Optimized Workflow**: Custom configurations to streamline development tasks.
